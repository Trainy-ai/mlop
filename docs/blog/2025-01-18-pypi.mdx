The first version of **MLOP** is now officially [published on PyPI](https://pypi.org/project/mlop/).

{/* truncate */}

You can install it via Python's built-in package manager, pip:

```bash
pip install mlop
```

:::tip
We also included a "dev" variant which supports development features such as using a text-based keyring as an alternative authentication backend.

You may specify this variant using

```bash
pip install "mlop[dev]"
```
:::
